# Harness Operations Notes

The harness is a documentation and process repository. Operational concerns apply primarily to projects that import it.

Project operational planning should cover:

- environments
- deployment process
- configuration
- logs, metrics, and traces
- dashboards and alerts
- runbooks
- incident response
- rollback and recovery
