# Accessibility Standards

## Baseline

Default target: WCAG 2.2 AA unless a project explicitly defines another standard.

## Required Practices

- Use semantic structure.
- Ensure keyboard operation.
- Provide visible focus indicators.
- Maintain sufficient color contrast.
- Label form fields and interactive controls.
- Make error messages clear and associated with inputs.
- Avoid motion that cannot be paused or reduced.
- Preserve logical reading and focus order.

## Validation

- Run automated checks.
- Test keyboard-only flows.
- Spot-check screen reader output for critical paths.
- Test responsive layouts.
- Record known gaps and remediation plans.
