# Code Review Checklist

Review for:

- [ ] requirement alignment
- [ ] correctness
- [ ] edge cases
- [ ] failure modes
- [ ] security and privacy
- [ ] accessibility
- [ ] reliability
- [ ] observability
- [ ] maintainability
- [ ] test quality
- [ ] documentation
- [ ] operational impact
- [ ] rollback impact

Review comments should be specific, actionable, and tied to risk or requirement impact.
