# Definition Of Ready

An implementation increment is ready to begin when:

- [ ] Project scale is classified.
- [ ] Relevant steering assets are selected.
- [ ] Requirements are approved.
- [ ] Acceptance criteria are testable.
- [ ] Architecture is approved or the change is explicitly architecture-neutral.
- [ ] Dependencies and constraints are identified.
- [ ] Risks are captured.
- [ ] Required tests are identified.
- [ ] Documentation impacts are understood.
- [ ] Human gate approval has been recorded.
